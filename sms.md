# SMS-сообщения

_[Вернуться к оглавлению](readme.md)_

## `MRIM_CS_SMS` (0x1039) C -> S

Запрос на отправку SMS-сообщения.

| Тип данных | Примерное содержание | Значение          |
| ---------- | -------------------- | ----------------- |
| UL         | N/A                  | Флаги сообщения (можно оставить 0) |
| LPS        | `+78005553535`       | Номер телефона (в ASCII)           |
| LPS        | `Hello world!`       | Сообщение                          |

Кодировка сообщения зависит от клиента, она может быть Windows-1251 или UTF-16LE.

## `MRIM_CS_SMS_ACK` (0x1040) S -> C

Посылается сервером в ответ на `MRIM_CS_SMS` с кодом ошибки.

| Тип данных | Примерное содержание | Значение          |
| ---------- | -------------------- | ----------------- |
| UL         | N/A                  | Код ошибки        |

Возможные коды ошибки:

- 0x01 - Доставлено
- 0x02 - Сервис недоступен
- 0x10000 - Неверные параметры
